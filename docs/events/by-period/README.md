# 시기별 이벤트 정본 문서

이 디렉토리는 이벤트 정본 문서를 생애 단계 기준으로 나눠 관리한다.  
운영자가 실제 발화 시점과 생애 흐름을 따라 읽기 좋도록 시기별 분류를 기준으로 한다.

## 시기 목록

- [유년기 / 아동기](./유년기-아동기/README.md)
- [청소년기](./청소년기/README.md)
- [진로 / 직업기](./진로-직업기/README.md)
- [중년기](./중년기/README.md)
- [노년기](./노년기/README.md)

## 운영 메모

- 이벤트 상세 정본은 모두 이 디렉토리 계층 아래에만 둔다.
- 체인 연결이 있더라도 파일 위치는 바꾸지 않는다.
- 체인 흐름을 볼 때는 [체인 허브](../chains/README.md)를 함께 참조한다.
