# 장기자랑 성공

## 개요

장기자랑 체인의 성공 후속 이벤트다.  
17세에 무대에 섰고 외모 티어가 높을 때만 열린다.

## 기본 정보

- 코드: `talent_show_result_good`
- 표시 문구: `무대에 서는 순간 모두의 시선이 집중됩니다.`
- 확률: `1.0`
- 원본 정의 파일: `events.js`

## 발화 조건

- 나이가 `18세`여야 한다.
- `eventState.choiceByCode.talent_show_17 === 'perform'`여야 한다.
- 외모 티어가 `SSR` 또는 `SR`이어야 한다.

## 선택지와 결과

- `ok` / `모두의 이목을 끌었습니다! 당신은 인기인!`
  - `resultText`: `모두의 이목을 끌었습니다! 당신은 인기인!`
  - `result`: `trait_delta(trait='app', delta=1)`
  - `trait_delta`는 이벤트 정의의 축약형 payload 그대로 `trait='app', delta=1`이 기록되어 있다.

## 운영 메모

- 외모 티어 상위 캐릭터에게만 보상이 들어간다.
- 청소년기에서 외모 티어 체감이 가장 분명한 후속 이벤트다.
