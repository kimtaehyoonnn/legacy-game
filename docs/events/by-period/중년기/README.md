# 중년기 이벤트

## 개요

이 시기 문서는 40대 후반부터 50대 후반까지의 전환기 이벤트를 다룬다.  
중년 재설계, 대형 경제 변동, 소비, 질병 진입이 함께 배치된다.

- [감기 발생 (`catch_cold`)](./감기-발생_catch_cold.md)

## 운영 포인트

- `buy_lottery_ticket`는 단일 이벤트 기준 최대 자산 변동을 만든다.
- `go_on_trip`, `catch_cold`는 후반 건강/질병 축의 진입점으로 자주 함께 본다.
