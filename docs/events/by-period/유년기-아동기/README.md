# 유년기 / 아동기 이벤트

## 개요

이 시기 문서는 5세부터 12세까지의 초기 서사 이벤트를 다룬다.  
초반 성향 형성, 첫 감정 이벤트, 첫 분기 체인이 집중된 구간이다.
