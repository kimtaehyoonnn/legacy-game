# 수업 중 급뇨

## 개요

8세 시점의 코믹하지만 페널티가 있는 이벤트다.  
선택에 따라 대인 성향이 직접 깎일 수 있다.

## 기본 정보

- 코드: `class_urgent_pooping`
- 표시 문구: `초등학교 수업 시간에 갑자기 오줌이 마렵다.`
- 확률: `0.8`
- 원본 정의 파일: `events.js`

## 발화 조건

- 나이가 `8세`여야 한다.
- 다른 추가 조건은 없다.

## 선택지와 결과

- `wet` / `바지에 지린다.`
  - `resultText`: `반 친구들이 모두 보고 말았습니다. 대참사가 발생했습니다.`
  - `result`: `trait_delta(trait='per', delta=-2)`
  - `trait_delta`는 이벤트 정의의 축약형 payload 그대로 `trait='per', delta=-2`가 기록되어 있다.
- `hold` / `용기를 내서 참아본다.`
  - `resultText`: `간신히 참아냈습니다.`
  - `result`: `none`
  - 실제 `result`는 `none`이며, 즉시 바뀌는 상태값은 없다.
- `drip` / `조금씩 내보내며 버틴다.`
  - `resultText`: `조절에 실패했습니다. 대참사가 발생했습니다.`
  - `result`: `trait_delta(trait='per', delta=-1)`
  - `trait_delta`는 이벤트 정의의 축약형 payload 그대로 `trait='per', delta=-1`이 기록되어 있다.

## 후속 연결

- 이전 이벤트: 없음
- 다음 이벤트: 없음
- 같은 체인 문서: 없음
- 같은 시기 인덱스: [유년기 / 아동기 인덱스](./README.md)

## 운영 메모

- 저연령 구간에서 비교적 강한 페널티를 주는 편이라 체감이 크다.
- 후속 체인은 없지만, 캐릭터 인상에 남는 초반 이벤트 역할을 한다.
