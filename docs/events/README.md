# 이벤트 문서 허브

이 디렉토리는 이벤트 운영 문서를 `시기별 정본 문서`와 `체인 허브 문서`로 나눠 관리한다.  
개별 이벤트 상세를 볼 때는 시기별 문서로 들어가고, 분기 흐름을 파악할 때는 체인 문서를 본다.

## 운영 원칙

- 정본 이벤트 문서는 모두 `by-period` 아래에만 둔다.
- 체인 문서는 링크 허브이며, 개별 이벤트 본문을 중복하지 않는다.
- 파일명은 `한국어-제목_{eventCode}.md` 규칙을 유지한다.
- 개별 문서는 한국어 본문으로 작성하고, 이벤트 코드와 원본 파일을 명시한다.
