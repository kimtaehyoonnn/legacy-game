# 장기자랑 체인

## 개요

장기자랑 체인은 17세 공연 선택이 18세 성공/실패 후속 이벤트로 이어지는 구조다.  
분기 기준은 `eventState.choiceByCode.talent_show_17`와 외모 티어(`traits.app.tier`)다.

## 흐름

`talent_show_17`에서 `perform` 선택  
-> `talent_show_result_good` 또는 `talent_show_result_bad`

## 분기 조건 요약

- `perform`를 고른 경우에만 후속 이벤트가 열린다.
- 외모 티어가 `SSR/SR`이면 성공, 아니면 실패 루트로 간다.
