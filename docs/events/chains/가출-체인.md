# 가출 체인

## 개요

가출 체인은 15세 중2병 이벤트에서 `runaway`를 고르면 16세 위기 이벤트로 이어지는 구조다.  
선택지 하나가 생존 리스크로 연결되는 대표적인 체인이다.

## 흐름

`middle_school_chuunibyou`에서 `runaway` 선택  
-> `runaway_danger_16`

## 분기 조건 요약

- 15세 이벤트에서 `runaway`를 고른 경우에만 후속 이벤트가 열린다.
- 조건식은 `eventState.choiceByCode.middle_school_chuunibyou === 'runaway'`다.

## 관련 이벤트

- [중2병 (`middle_school_chuunibyou`)](../by-period/청소년기/중2병_middle_school_chuunibyou.md)
- [가출 후 위기 (`runaway_danger_16`)](../by-period/청소년기/가출-후-위기_runaway_danger_16.md)

## 구현 메모

- `runaway_danger_16`의 `follow` 선택은 `혼절`로 이어져 매우 치명적이다.
- 체인 길이는 짧지만 페널티 강도는 높은 편이다.
