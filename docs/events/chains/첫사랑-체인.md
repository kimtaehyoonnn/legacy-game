# 첫사랑 체인

## 개요

첫사랑 체인은 11세의 선택이 12세 후속 이벤트 분기로 이어지는 구조다.  
분기 기준은 `eventState.choiceByCode.first_crush`와 외모 티어(`traits.app.tier`)다.

## 흐름

`first_crush`  
-> `first_crush_confess_good` 또는 `first_crush_confess_bad`  
-> `first_crush_tsundere_good` 또는 `first_crush_tsundere_bad`

## 분기 조건 요약

- `confess`를 고르면 고백 루트로 간다.
- `tsundere`를 고르면 츤데레 루트로 간다.
- 12세 후속 이벤트는 외모 티어가 `SSR/SR`인지 여부에 따라 성공/실패가 갈린다.
